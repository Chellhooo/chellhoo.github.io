  1. 使用<!–more–>显示出错

    [背景] 有时候我们会对文章增加阅读全文的功能，在非正文阅读的页面只显示文章部分文字，对此我们有两个方法

    • 在相关主题的themes/主题名/_config.yml中设置auto_excerptenable开关为true
    • 在文章中间加上<!–more–>，则文章该处后面的文字将会隐藏
