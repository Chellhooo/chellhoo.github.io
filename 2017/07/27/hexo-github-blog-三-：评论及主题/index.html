评论功能

之前那为大家所熟知可提供评论支持的多说和网易云的跟帖都相继下线，感觉如今硕果仅存的就只剩下畅言 (:з」∠)，希望它还能坚挺住才好。
言归正传，给hexo添加评论功能，这次是以next主题为例子。

  1. 注册畅言账号

    【注册】->新建站点->鼠标移到左侧导航栏，滚动条滚到最后点击【评论插件】->点击【评论管理】->点击左侧导航栏中【后台总览】->【appid】&【appkey】get！
    • 备案
      很多人可能也会看到[网站使用畅言必须提供备案号，否则只能使用15天]的提示，鉴于这是挂载在github上面的博客，不少人并没有购买并对其绑定自己的域名，所以如果需要增加评论的功能就要花钱注册域名并进行备案了哦 |·_·)

  2. 配置

    新版的next主题已经集成了畅言评论插件，只需要打开themes/next/_config.yml找到畅言的配置你的畅言appid和appkey就ok啦。
