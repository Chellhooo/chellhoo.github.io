hexo+github blog：小坑集合

Posted on 2017-07-28 | In hexo blog

  1. 使用<!–more–>按钮显示weiterlesen

    [解决方法] 在themes/next/scripts/下创建 excerpt.js，把下列代码写进文件并保存,重新开启服务器，就可以看到已经恢复正常了
    [参考链接] https://stackoverflow.com/questions/32933052/hexo-excerpt-more-not-working-after-update

    [解决方法] 在_config.yml配置除德文(themes/主题名/languages/de.yml是德文配置文件)以外的语言！
    是的你没猜错，weiterlesen是一句德文 ：）
    [意外收获] 使用hexo s --debug可以知道hexo是否有解析到我们所需的脚本文件

  2. 修改配置文件在本地总是看不到效果

    先别着急地崩溃，也先别着急检查步骤是否错了，先检查修改配置文件后是否有执行hexo deploy或者hexo generate
    • 只有新建、删除、修改文章才无需部署直接开启服务器即可在本地预览，待效果满意之后再hexo d -g部署上github
